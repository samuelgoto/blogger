# blogger

1. git clone
  * git clone https://github.com/samuelgoto/blogger.git
2. add dependencies
  * git submodule init
  * git submodule update
3. start a local server
  * python -m SimpleHTTPServer
4. Run automated tests
  * http://localhost:8000/tests/all_tests.html
5. Build a release
  * ./build.sh
